Global Function

toRadians()

Declaration

func toRadians(degrees: Double) -> Double

Parameters

degrees degree value to be converted to radians.

Returns

Double

Description

Convert a value in degrees to its corresponding value in radians. There are 360 degrees in a circle, and 2 * PI radians in a circle.

Examples

{% highlight swift %} let valueInDegrees = 90.0 let valueInRadians = toRadians(degrees: valueInDegrees) {% endhighlight %}
