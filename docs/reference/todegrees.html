Global Function

toDegrees()

Declaration

func toDegrees(radians: Double) -> Double

Parameters

radians radian value to be converted to degrees.

Returns

Double

Description

Convert a value in radians to its corresponding value in degrees. There are 360 degrees in a circle, and 2 * PI radians in a circle.

Examples

{% highlight swift %} let valueInRadians = M_PI let valueInDegrees = toDegrees(radians: valueInRadians) {% endhighlight %}
