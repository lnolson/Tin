Global Function

nanosecond()

Declaration

func nanosecond() -> Int

Parameters

None

Returns

Int

Description

Current nanosecond returned by system clock. An Int value in the range 0 to 999999999. A nanosecond is one billionth of a second. eg. 1 / 1,000,000,000 of a second.

Examples

{% highlight swift %} var nano = nanosecond() {% endhighlight %}
