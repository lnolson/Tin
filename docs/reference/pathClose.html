Global Function

pathClose()

Declaration

func pathClose()

Parameters

None.

Returns

-

Description

Use pathBegin and pathEnd to create a shape using a series of points. Use pathClose to make a new segment that connects the current point with the first point on the path.

Examples

{% highlight swift %} // Create a closed triangle with 3 points and pathClose pathBegin() pathVertex(x: 50.0, y: 50.0) pathVertex(x: 125.0, y: 100.0) pathVertex(x: 50.0, y: 150.0) pathClose() pathEnd() {% endhighlight %}
