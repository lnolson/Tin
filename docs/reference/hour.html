Global Function

hour()

Declaration

func hour() -> Int

Parameters

None

Returns

Int

Description

Current hour returned by system clock. The hour value returned is an Int, in the range 0 to 23. For example, during the 4PM hour, the value returned is 16.

Examples

{% highlight swift %} var h = hour() {% endhighlight %}
