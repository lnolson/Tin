Class

TFont

Overview

An object to represent a font.

Properties

var horizontalAlignment: TinFontHorizontalAlignment

Default value is .center.

var verticalAlignment: TinFontVerticalAlignment

Default value is .baseline.

var lineHeightMultiple: CGFloat

Default value is 1.0.

var paragraphAlignment: NSTextAlignment

Default value is .left.

Initializers

init(fontName: String, ofSize size: CGFloat)

Initializes and returns the font object using the font with the name of fontName, at the specified size.

Methods

func size(ofMessage message: String) -> NSSize

Calculate the size of a String if it was rendered with the current font settings.
