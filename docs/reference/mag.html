Global Function

mag()

Declaration

func mag(x: CGFloat, y: CGFloat) -> CGFloat

Parameters

x First vector component.
y Second vector component.

Returns

CGFloat

Description

Calculates the magnitude of a (2d) vector. This vector is relative to the origin (0,0).

Examples

{% highlight swift %} let x = 150.0 let y = 180.0 let m = mag(x: x, y:y) // The value of m is 234.3074 {% endhighlight %}
